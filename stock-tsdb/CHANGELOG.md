# Stock-TSDB 变更日志

所有 Stock-TSDB 项目的显著变更都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [未发布]

### 新增
- 完善项目文档、脚本和Makefile
- 新增业务数据Web服务器启动脚本
- 新增SQL查询字段顺序修复功能
- 新增生产环境监控脚本

- 优化Makefile构建目标，增加更多测试选项
- 改进脚本的错误处理和日志输出
- 完善项目文档结构

## [1.0.0] - 2024-12-01

### 新增
- V3存储引擎基础版本和集成版本
- 业务数据分离架构
- Consul集群集成支持
- Redis TCP服务器兼容性
- 冷热数据存储功能
- 完整的监控和告警系统

### 功能
- 高性能时序数据存储和查询
- 支持SQL-like查询语法
- 集群部署和负载均衡
- 数据压缩和缓存优化
- 实时监控和健康检查

## [0.9.0] - 2024-11-15

### 新增
- 基础存储引擎实现
